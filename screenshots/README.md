# Screenshots

Place Chrome Web Store screenshots here before submission. Required: at least 1 at 1280x800 or 640x400; max 5. Capture direction is in `docs/STORE-LISTING.md`.

Suggested filenames:
- `01-hero-grid-with-button.png` : a board view with the red copy button visible at the bottom-left of a pin on hover.
- `02-image-copied-toast.png` : the same board view with the 'Image copied' toast.
- `03-options-page.png` : the settings page.
- `04-detail-video.png` : a video pin's detail view.
- `05-native-vs-extension.png` : optional before/after comparison.
